Sokoban

Tutorial

Get all the movable blocks in the goal positions.

You can only push blocks away from you onto empty spaces.

If you get stuck you can reset the game and try again.

R key to reset the game.

Up Arrow or W key to move up.

Down Arrow or S key to move down.

Right Arrow or D key to move right.

Left Arrow or A key to move left.
